What is this?

TG Photo Backup is a lightweight, open-source alternative to Google Photos. It uses your own Telegram bot to send photos directly to your Telegram chat ‚Äî both in compressed and original quality.

Features

  • Unlimited photo backup via Telegram
  • Supports photos up to 50MB
  • Only syncs new or unbacked photos
  • Supports multiple folder paths
  • Backs up settings and photo metadata

How to Use

  1. Install the APK on your Android phone
  2. Manually grant file and media permissions
  3. Configure your Bot API key and Chat ID in the app
  4. Set folder paths to include/exclude
  5. Use the Backup or Sync option to upload

Telegram Bot Setup

1. Create a bot using @BotFather

2. Get your Telegram user ID from @Check_Telegram_IDBot

3. Add both to the app settings
